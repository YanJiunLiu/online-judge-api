## Installation
#### 1. Environment 
```
    OS: macos M3
    Python version: v3.13
    Ollama version: 0.11.7
    uv version: 0.8.13
```
#### 2. Uv installation
```bash
    brew install uv
    uv init llm --name llm --author-from git --description "LangChain Project Development" -p 3.13
```
